- Vue-Cli와 Vue-router를 툴로 이용하였습니다.

- 가장 최근에 대화한 친구가 목록 최상단에 위치하게 설정하였습니다.
- 메세지를 보내면, 일정한 시간 뒤 해당 메세지의 글자 배열을 랜덤하게 섞어 다시 반환하게 설정하였습니다.
